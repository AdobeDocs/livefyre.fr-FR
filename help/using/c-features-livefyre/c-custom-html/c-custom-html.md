# Code HTML personnalisé{#custom-html}

Autorisez les modérateurs, propriétaires et blogs à ajouter des pièces jointes HTML personnalisées au contenu publié via les commentaires ou le blog en direct.

La fonction Incorporation HTML personnalisée permet aux modérateurs, propriétaires et blogs d&#39;ajouter des pièces jointes HTML personnalisées au contenu qu&#39;ils publient via les commentaires ou le blog en direct. Cette fonctionnalité peut être utilisée pour joindre n&#39;importe quel contenu à vos publications, y compris les lecteurs vidéo ou les widgets d&#39;interrogation propriétaires, en plus des images standard, des lecteurs vidéo et d&#39;autres éléments incorporés automatiquement par les applications Livefyre.

Collaborez avec votre gestionnaire de compte pour activer cette fonctionnalité pour votre réseau ou votre site.

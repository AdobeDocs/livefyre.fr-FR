# Balises de collection et de contenu {#collection-and-content-tags}

Ajoutez des balises de mots-clés aux collections et au contenu, afin de classer et d’organiser le contenu pour faciliter la récupération.

L’affectation de balises permet de filtrer les recherches selon une balise spécifique et d’extraire les données de l’API d’index de chaleur.

Les balises peuvent être ajoutées via les panneaux **[!UICONTROL Content]** et **[!UICONTROL Collections]** dans Studio, ou directement dans les métadonnées de la collection (à l’aide de l’objet collectionMeta) lors de l’initialisation ou de la mise à jour de la collection avec Livefyre.

## Utilisation des balises de collection avec l’API d’index de chaleur {#section_uhd_frk_wy}

L’API Heat Index accepte un paramètre de balise, ce qui vous permet de récupérer la chaleur relative de toute collection avec cette balise, sur un site ou sur l’ensemble du réseau.

Pour plus d’informations sur l’implémentation, reportez-vous à la documentation technique de l’API Heat Index.

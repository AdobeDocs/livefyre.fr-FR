# Feature Card{#feature-card}

Ajoutez des éléments uniques de contenu incitatif à votre page.

Utilisez la carte de caractéristiques pour mettre en évidence des éléments de contenu individuels sur un seul écran de carte. L’application donne la priorité au contenu proposé le plus récemment et passe en revue les nouveaux éléments afin d’assurer un nouvel affichage pertinent.

La carte des fonctionnalités affiche le dernier élément présenté (pas nécessairement le dernier élément publié), ce qui vous permet de sélectionner votre meilleur contenu pour une présentation opportune sur la page.
